Proyecto de Grado

Este trabajo de grado se enfoca en la creación de un prototipo web para un sistema de recomendación de compra de motocicletas personalizado. El objetivo principal es proporcionar a los usuarios sugerencias de compra basadas en sus necesidades individuales, considerando factores como el uso previsto, preferencias de estilo y características técnicas. Este sistema busca mejorar la experiencia de compra de motocicletas, brindando recomendaciones precisas y relevantes que se ajusten a las expectativas y requerimientos específicos de cada usuario.
